# Daily Tracker (Python)

A beginner-friendly terminal project built in Python to practice loops, lists, conditions, and user interaction.

## Features

✅ Enter user name
✅ Add tasks
✅ Complete tasks
✅ View remaining tasks
✅ Exit program
✅ Handles invalid menu choices

---

## How It Works

When the program starts:

1. User enters their name
2. A menu appears
3. User chooses actions

Menu:

1 → Add Task
2 → Complete Task
3 → View Tasks
4 → Exit

---

## Example

Enter your name:

---

## Important Usage Note

Add ONE task at a time.

❌ Incorrect:

Enter task:
Python, German, Sketch

(This becomes one single task.)

✅ Correct:

Enter task:
Python

Then add another task separately.
